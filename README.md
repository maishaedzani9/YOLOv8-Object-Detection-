YOLOv8 Object Detection and Tracking

   Overview
This project demonstrates real-time object detection and tracking using the YOLOv8 deep learning model.

The system detects objects in a video and tracks them across frames using the ByteTrack algorithm.

This project was developed as part of the  Artificial Intelligence project.

   Features
- Object detection using YOLOv8
- Multi-object tracking in video streams
- Real-time webcam detection
- Automatic output saving

Technologies Used
- Python
- Ultralytics YOLOv8
- OpenCV
- PyTorch
-  Project Workflow

1. Load YOLOv8 pretrained model
2. Process input source (image/video/webcam)
3. Detect objects in each frame
4. Track objects across frames
5. Save processed output

   Example Detected Objects
The system successfully detected objects such as:

- Cars
- Trucks
- Trains
- Peopl
